How It Works

Importing Images to Salesforce

This part of the tool automates the creation of a ZIP archive for bulk importing images into a Salesforce Enhanced CMS Workspace. For each image you upload, it creates a folder named after the image. Inside each folder, it generates the required JSON files and a _media subfolder for the image itself.

salesforce_import.zip/
└── LMH50565/
    ├── _media/
    │   └── LMH50565.jpg    (Your image file)
    ├── content.json        (Content details)
    └── _meta.json          (Folder and metadata)

Exporting Data from Salesforce

Upload a .zip file exported from Salesforce. The tool will find every content.json file and extract key information. If you provide your org details, it will also construct a full Public URL for each media item.
