How It Works

This tool automates the creation of a ZIP archive for bulk importing images into a Salesforce Enhanced CMS Workspace. It follows the specific folder and file structure required by Salesforce.

1. Upload Images

Drag and drop your image files (e.g., LMH50565.jpg) into the upload area above, or click to select them from your computer.

2. Specify CMS Folder

Enter the name of the destination folder in Salesforce where you want the images to appear after import. This is used in the _meta.json file.

3. Generate ZIP

Click the "Generate and Download ZIP" button. The tool will create a .zip file in your browser that you can directly upload to Salesforce.

Generated ZIP Structure

For each image you upload, the tool creates a folder named after the image. Inside each folder, it generates the required JSON files and a _media subfolder for the image itself.

salesforce_import.zip/
└── LMH50565/
    ├── _media/
    │   └── LMH50565.jpg    (Your image file)
    ├── content.json        (Content details)
    └── _meta.json          (Folder and metadata)

JSON File Contents

The JSON files are configured as follows:

content.json

{
  "type": "sfdc_cms__image",
  "title": "LMH50565",
  "contentBody": {
    "sfdc_cms:media": {
      "source": {
        "type": "file",
        "mimeType": "image/jpeg"
      }
    }
  }
}

_meta.json

{
  "path": "Product Images"
}
